Cara Buat RifqyDesign

Web portfolio desain minimalis — 2 menit jadi!

  1. Buat repo GitHub baru → RifqyDesign
  2. Buat folder works → masukkan semua hasil karyamu (jpg/png)
  3. Copy file index.html (yang utama) → paste ke repo
  4. Ganti gambar di folder works/1.jpg, 2.jpg, dst
  5. Ganti judul & deskripsi sesuka hati
  6. Enable GitHub Pages di Settings → pilih branch main & folder /root
  7. Selesai! Link: rydevs29.github.io/RifqyDesign

Simple, cepat, dan langsung bisa dipamerin!
